# Shogi (Japanese Chess)
Variants of Shogi written in Julia programming language. 

## Features
- GUI
- Time Settings
- Roulette
- Multiple variants of the game (standard shogi, minishogi, chu shogi, tenjiku shogi)
- Multiple AI difficulties (easy, medium, hard)
- networking
